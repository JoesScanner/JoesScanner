# JoesScanner App

JoesScanner is a cross-platform app that lets you listen to live radio calls from the Joe’s Scanner service or from your own Trunking Recorder server. It provides a clean, simple interface with reliable audio streaming and automatic call handling.

The goal of this app is to make real-time radio monitoring easy and accessible on Windows, Android, iOS, and macOS.

---

## What the App Does

- Plays radio calls as they come in  
- Shows incoming calls in a queue  
- Lets you jump to live activity  
- Adjusts playback speed  
- Shows connection status  
- Allows using Joe’s Scanner or your own server  
- Saves your settings automatically  
- Includes a log viewer if you need to troubleshoot  

The app is built for continuous, long-term listening.

---

## Using With Joe’s Scanner

If you are a Joe’s Scanner subscriber, the app can connect directly to the service.

Just enter the server URL and your login credentials (required).  
The app will automatically begin loading calls and streaming audio.

---

## Using Your Own Trunking Recorder Server

If you run your own TR setup (Trunking Recorder), the app can connect to it as well.

You will need:

- Your server URL  
- A username  
- A password (if enabled)  

Once entered in the Settings screen, the app will load your call list just like Joe’s Scanner.

---

## Basic Setup

1. Install the app for your platform  
2. Open the Settings screen  
3. Enter your server URL  
4. (Optional) Enter your username and password  
5. Press “Test Connection”  
6. Save your settings and return to the main screen  

If everything is correct, calls will begin displaying automatically.

---

## Common Features

### Call Queue  
The app keeps track of calls waiting to be played and shows them in order.

### Audio Controls  
- Play the latest call  
- Jump to live activity  
- Adjust playback speed  

### Connection Status  
A badge shows whether the app is currently connected.

### Logging  
You can open the built-in log viewer to see connection details or export the log if support is needed.

---

## Troubleshooting

If calls are not appearing or audio is not playing:

- Verify the server URL  
- Make sure your username/password are correct  
- Try the “Test Connection” button  
- Check the log viewer for connection errors  
- Ensure your server is reachable from your network  

If using Trunking Recorder, make sure the Call History API is enabled and reachable.
